▲ Commands Worth Knowing

/schedule Create a recurring Claude Code routine that runs on Anthropic's infrastructure — no local machine required. New this week.
/mcp List and manage active MCP server connections in Claude Code. Use to verify Slack, Notion, and BigQuery connectors are live before a session.
/doctor Run Claude Code's diagnostics to check plugin health, MCP server status, and session configuration. First stop when something stops working.
gh run watch Stream live output from a GitHub Actions run in the terminal. Eliminates browser tab flipping when waiting on a CI pipeline.
/compact Compress the current conversation to free context window space without losing key state. Run before long agentic sessions.
bq query Run a BigQuery SQL query directly from the terminal (bq query --use_legacy_sql=false 'SELECT ...'). Pair with Claude Code for one-shot data analysis without opening the console.
