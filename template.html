Commands Worth Knowing

/resume Pick up your last Claude Code session exactly where you left off — context, files, and tool state intact.
/schedule Create a scheduled remote agent that runs on a cron. The newsletter you're reading was built by one.
/review Run a full code review on the current pull request. Catches issues before your teammates have to.
/init Generate a CLAUDE.md file for your project — gives Claude persistent context about your codebase and conventions.
/context Show how much of the context window you've used and what's consuming it. Essential for long sessions.
/compact Compress the conversation to free up context space without losing key information. Use before you hit the wall.
