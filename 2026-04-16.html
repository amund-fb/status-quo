⌘ Commands Worth Knowing

claude /schedule Set up a Claude Code Routine that runs on a cron — no local machine required. The newsletter you're reading was generated by one.
bq query --nouse_legacy_sql Run standard SQL against BigQuery from the terminal. Pair with Claude Code to iterate on financial queries without leaving the shell.
gh run watch Stream a GitHub Actions workflow in real time from your terminal. Know when your deploy finishes without refreshing a browser tab.
claude --resume Pick up your last Claude Code session exactly where you left off — context, files, and tool state intact.
mcp list See every MCP server connected to your current session. First thing to check when a tool silently fails.
npx live-server . Instant local server with hot reload. No config. The fastest way to preview any static HTML file.
